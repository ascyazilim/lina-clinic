# Lina Clinic Web

Lina Guzellik Merkezi icin React + TypeScript + Vite tabanli frontend iskeleti.

## Teknolojiler

- React
- TypeScript
- Vite
- Material UI
- Material Icons
- React Router DOM
- Axios
- MUI X Date Pickers
- Dayjs

## Komutlar

```bash
npm install
npm run dev
npm run build
```

## Ortam Degiskeni

`.env`

```env
VITE_API_BASE_URL=http://localhost:8080
```

## Route Yapisi

- `/`
- `/hizmetler`
- `/hizmetler/:slug`
- `/randevu`
- `/hakkimizda`
- `/iletisim`
- `/kvkk`
- `/admin/login`
- `/admin/dashboard`
- `/admin/randevular`

## Not

Bu adimda proje kurulumu, routing, tema, layout yapisi, bos sayfalar ve Axios istemcisi hazirlandi.
Gercek randevu formu ve admin randevu tablosu sonraki adimda gelistirilecektir.
